Player 🛠 Builder

Put a presentation asset (video, deck HTML/PDF, subtitle file…) onto the decentralized web, then paste the resulting ipfs:// or magnet: reference into a source in the Builder. Everything here runs in your browser — no p2present server sees your files or tokens. See the hosting guide for the full story (plain URLs, CLI seeding, etc.).

📌 Pin to IPFS

Upload a file to a pinning provider you configure with your own API token. Returns a CID and a ready ipfs:// reference.

🌐 Seed via WebTorrent

Create a torrent in the browser and seed it from this tab. Copy the magnet into a manifest. Seeding stops when you close this tab — for always-on seeding, use the CLI (see the guide).

📦 Hosted references

References produced in this browser. Copy one and paste it into a video / deck / subtitle source in the Builder.
