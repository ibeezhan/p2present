Open source

Fork it. Host it. Keep it.

p2present is a tiny, dependency-light static site under the MIT license. No server runtime to pay for, nothing to lock you in — clone the repo, drop your content into docs/content/, enable GitHub Pages, and you have your own instance on your own domain.
